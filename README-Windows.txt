QuietPatch for Windows

1) Install Python 3.11 (64-bit) from python.org.
2) Double-click run_quietpatch.bat.
3) Reports will be written to .\reports\quietpatch_report.html (or as configured).

If you see an error about the Python launcher, install Python 3.11 (64-bit) and run again.

Files in this folder:
- quietpatch-win-py311.pex
- run_quietpatch.bat
- catalog/ (if provided)
- policies/ (if provided)
- README-Windows.txt
