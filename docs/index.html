QuietPatch 🔐

Privacy-first vulnerability scanner. Offline. Cross-platform. No telemetry. No auto-patching.

Install (macOS & Linux)

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Matt-C-G/QuietPatch/main/install.sh)"

Install (Windows PowerShell)

irm https://raw.githubusercontent.com/Matt-C-G/QuietPatch/main/install.ps1 | iex

Run a scan

quietpatch scan --also-report --open

Opens report.html in your browser. Works fully offline with the bundled database.

  • 📦 App inventory
  • 🛡️ CVEs with severity badges (Critical/High/Medium/Low)
  • 🚨 KEV & EPSS surfaced
  • 🔧 Concrete remediation commands (copy-to-clipboard)
  • 📑 Deterministic, reproducible output

Verify downloads

We publish checksums for every release. Verify like this:

shasum -a 256 -c SHA256SUMS

Optional: verify signed checksums with Minisign (see VERIFY.md).

FAQ

Does QuietPatch change my system?

No. It’s read-only. Remediation commands are suggestions for you to run manually.

Does it need the internet?

No. Use the offline DB snapshot included by the installer. You can refresh with quietpatch update-db.

Which platforms?

macOS (Apple Silicon), Linux x86_64, Windows x64. Requires Python 3.11–3.13 (Windows bundles coming later).
